## タスク: Wikiの整合性チェック

以下の項目をすべてチェックし、レポートを生成してください。

### チェック項目

1. **Orphanページ**: index.mdに記載があるが、他のどのページからもリンクされていないページ
2. **リンク切れ**: `[[存在しないページ]]` へのリンク
3. **spoiler_scope不整合**: 参照先ページよりspoiler_scopeが小さいページ（ネタバレ漏れの可能性）
4. **空ページ**: フロントマターのみで本文がないページ
5. **Timeline抜け**: Eventページが存在するのにtimeline.mdに記載がないもの
6. **関係性の片方向**: AのページにBへのリンクがあるのにBのページにAへのリンクがない
7. **未作成ページ候補**: 本文中で複数回言及されているが独立ページがない概念やキャラクター
8. **伏線の棚卸し**: summariesの「伏線」セクションで「→ 未回収」のままになっているもの一覧
9. **フロントマター不備**: 必須フィールドが欠けているページ
10. **Index同期**: wiki/ 以下に存在するがindex.mdに未記載のページ

### レポート形式

チェック結果を以下の形式で出力する:

```
## Lint Report - [日付]

### 問題あり（要対応）
- [ ] 具体的な問題と修正方法

### 推奨（任意）
- [ ] 改善提案

### 統計
- 総ページ数: N
- Ingest済み話数: N / 139
- 未回収伏線数: N
```

### 自動修正

以下の問題は確認の上で自動修正する:
- Index同期の不整合 → index.mdを更新
- 関係性の片方向 → 不足側にリンクを追加

それ以外はレポートのみ。修正はユーザーの指示を待つ。

修正した場合は `log.md` に lint 操作として記録する。
